A tool made to check any Hypixel player's last session easily and quickly!

Session data courtesy of Slothpixel and Hypixel Inc. via their respective APIs.

This project is under the MIT license at https://github.com/Attituding
  • Stats will only show for Bed Wars, Duels, Blitz Survival Games, The Pit, Sky Wars, Speed UHC, UHC Champions, The Walls, and Mega Walls. More may be added later
  • Some accuracy will be lost after a while with the "Show the last user searched if the window closes" option as it doesn't actually make a request for new data
  • Only successful searches will be saved in your player search history tab. Any error like invalid usernames, fetch errors, etc will not be saved
