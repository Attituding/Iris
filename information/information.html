A simple tool to check a player's last session.

  • Stats will only show for the following gamemodes. More may be added later
    • Bed Wars, Duels, Blitz Survival Games, The Pit, Sky Wars, Speed UHC, UHC Champions, The Walls, Mega Walls
  • Some accuracy will be lost after a while with the "Keep the last user searched between refreshes" option as it doesn't actually make a request for new data
